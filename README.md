# EcoBin-X Smart Waste Management System

![EcoBin-X Logo](assets/icon.png)

## Overview
EcoBin-X is a **smart waste management** web application that monitors waste bin levels in real time using IoT sensors (NodeMCU, ultrasonic sensors). It provides:
- Real‑time bin level visualisation
- Automated SMS alerts when bins are near capacity (simulated with a fancy UI)
- Historical analytics via ThingSpeak
- A responsive gallery showcasing hardware, SMS proof, and the team
- A demo bin that always appears full for demo purposes

## Features
- **Live monitoring** of wet and dry waste levels
- **SMS alert simulation** with loading spinner, toast notification, and auto‑reset
- **Lightbox2** gallery for high‑quality images
- **Bootstrap 5** responsive layout with modern UI/UX (animations, glass‑morphism, dark mode support)
- **Demo bin** for presentations without live hardware

## Installation & Setup
1. **Clone the repository**
   ```bash
   git clone https://github.com/chaithanya-0414/EcoBin-X-Smart-Waste-Management-System.git
   cd EcoBin-X-Smart-Waste-Management-System
   ```
2. **Install dependencies** (only front‑end, no npm needed)
   - Ensure you have a modern browser (Chrome/Edge/Firefox).
   - No server is required – you can open `index.html` directly, or serve via a simple HTTP server for CORS safety:
     ```bash
     # Python 3 built‑in server (optional)
     python -m http.server 8000
     # then open http://localhost:8000 in your browser
     ```
3. **Configure Twilio (optional)**
   - Edit `sms.py` with your Twilio `ACCOUNT_SID`, `AUTH_TOKEN`, and `FROM`/`TO` numbers.
   - Run the script with Python to send real SMS alerts.

## Usage
- Open `index.html` for the landing page.
- Navigate to **Gallery** to view hardware photos.
- Use **Welcome** to see live bin monitoring and the demo bin.
- Click **Send SMS Alert** on a full bin to see the simulated sending flow.
- Visit **Achievements** to see the hackathon award.
- **Contact** page shows up‑to‑date contact details.
